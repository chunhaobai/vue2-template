# ai-assistant

这是一个 Vue2 项目的通用模板，包含以下内容：
- Vue2
- vue-router
- element-ui（全局引入）
- axios 封装
- SCSS 支持
- GtTable 二次封装表格

## 使用方式
```bash
git clone  git@github.com:chunhaobai/vue2-template.git my-new-project
cd my-new-project
npm install
npm run serve
